### AerialRobot
ELEC5660 Introduction to Aerial Robot project

Description: ROS, Minimum-snap trajectory, 3D-2D pose estimation with tag detector, Extended Kalman Filter(EKF) fusing IMU and tag odometry

Language: C++, Matlab
Library: OpenCV, Eigen
